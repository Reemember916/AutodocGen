功能概览

  • 从 C 源码生成 CSCI 详细设计 Word 文档（.docx）。
  • 支持单文件与工程模式，工程模式可按层级与自定义顺序生成。
  • 支持生成后更新 CSU 标识，并同步输出软件单元附表。
  • 支持 AI 辅助（可选），并提供日志、步骤进度与可续跑能力。

快速开始

  1. 选择模式：单文件 或 工程模式。
  2. 单文件：选择 C 文件；工程模式：选择 工程目录 并点击 加载。
  3. 设置 Word 输出 路径（.docx），可选模板文档。
  4. 点击右上角 生成文档，在右侧面板观察进度与日志。

工程模式下建议先加载工程，以便排序、合并模块等操作生效。

界面与区域说明

  • 侧边栏：主页 / 设置 / 帮助入口。
  • 顶栏：任务状态、进度条与运行按钮（生成/停止/继续/更新 CSU）。
  • 工作区：左侧输入卡片（源代码/输出），右侧面板（项目/步骤/日志）。

单文件 / 工程模式

单文件

  • 适合快速生成某个 C 文件的设计文档。
  • 只需要选择 C 文件与输出路径。

工程模式

  • 选择工程根目录（包含 SRC 目录）。
  • 加载后会按"应用层 / 中间层 / 驱动层"分类展示。
  • 支持拖拽排序、合并模块、导出单函数。

工程列表与模块管理

  • 右侧"项目"面板支持拖拽调整文件顺序（仅同一层内）。
  • 右键文件可 导出单函数（需选择函数名）。
  • 多选同层文件可 合并为模块；模块可重命名或解散。
  • 选中条目后按 Delete / Backspace 可从列表移除。

模块合并仅用于输出顺序与逻辑归类，不会改变源代码。

生成、停止与继续

  • 生成文档：开始生成，顶栏显示进度与估算时间。
  • 停止：在安全点停止并保留进度。
  • 继续：停止后可继续接力生成。

运行中会暂时禁用输入控件，避免任务参数被意外修改。

更新 CSU 标识

  • 选择已生成的 .docx 文件。
  • 点击 更新 CSU 标识，按源码顺序重排 CSU 标识。
  • 会优先沿用文档中已有的标识前缀（例如 D/R_SDD01_609）；若未识别到则使用"需求ID前缀"设置。
  • 工程模式下支持同步更新软件单元表与单元函数列表。

设置说明

基本

选项说明
需求ID前缀生成的需求编号前缀。
完成后自动打开生成后自动打开输出。
排除目录工程扫描时跳过的目录名称（每行一个）。
分层规则中间层/驱动层识别关键词（逗号分隔）。
术语表覆盖术语映射，用于命名优化。
符号字典覆盖精确符号映射，用于局部变量、全局变量、结构体成员、函数名、宏名的一致化。

AI

选项说明
AI无 AI / 开启 AI。开启后 AI 只作为受控表达增强，源码事实仍由规则、LSP 与 AST 约束。
调用方式本地、DeepSeek 或其他 OpenAI 兼容远程模型接口。
模型 / 固定地址模型名称从内网接口自动获取，调用地址固定为内网部署地址，无需 API Key。
Context(num_ctx)上下文长度（依模型而定）。
AI 并发数开启 AI 后用于函数级并发生成，默认 1。
Proxy可选代理。

性能

  • 工程扫描预筛选：跳过无注释/无函数文件，提高扫描速度。
  • 并行预处理线程：0 为自动。
  • AI 并发数：开启 AI 后的函数级并发窗口，建议从 1 开始调优。
  • 日志采样间隔：减少日志刷屏。

高级参数与术语表

术语表覆盖格式

  • 每行一条：KEY=中文
  • 或直接填写 JSON 字典：{"KEY": "中文"}

符号字典覆盖格式

  • 每行一条：symbol=中文
  • 或使用分区 JSON：{"functions":{"ReadCpuTimer1Counter":"读取CPU定时器1计数"},"members":{"frceRst":"强制复位"}}
  • 程序会先加载项目根目录下的 symbol_dictionary.json，再叠加此处覆盖项。
  • 生成时还会自动维护项目级 AI 记忆库 autodoc_symbol_memory.json，后续同项目会优先复用。

高级参数

常用项：

  • max_dist / min_ratio：AI JSON key 容错参数。
  • ai_retry_times：AI 失败重试次数。
  • ai_fail_policy：失败策略（fallback/skip_function/circuit_fallback）。
  • ai_regression_rounds：首轮效果不佳时的自动回归补跑轮数，默认 1。
  • ai_regression_force_one_call：回归补跑时是否强制启用 one_call，默认 1。

常见问题

提示"工程目录不存在 / 未找到 SRC"

请确认工程目录正确，且包含 SRC（不区分大小写）。

提示"输出文件必须是 .docx"

请在输出路径中选择 .docx 扩展名。

AI 调用失败或很慢

检查模型列表是否已刷新、固定内网地址是否可访问，或将 AI 模式切换为 0/1。

日志过多影响体验

可在"性能"中提高"日志采样间隔(N)"。
