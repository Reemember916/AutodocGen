## 4.2.7.2 左吊舱空中加油模式（PROJECT-2007R_SDTD_0023）

### 进入条件
1. 接收RIU“加受油模式=左吊舱”。

### 任务前状态检查（全部通过后进入加油流程）
2. 检查以下状态：
   - 压力加油控制活门关到位；
   - 三通阀在空中受油位/关闭位；
   - 应急放油切断阀在关位；
   - 连通阀在关闭位；
   - 电动通气阀在打开位；
   - 燃油测量系统正常。

2.a~2.f 故障处理（任一失败触发相应提示）：
- 压力加油控制活门未关到位：`XX油箱压力加油控制活门故障`。
- 三通阀到位异常：`三通阀故障`。
- 应急放油切断阀未到位：发关闭指令，5s内仍未到位 -> `左(或右)应急放油切断阀故障`。
- 连通阀未到位：发关闭指令，5s内仍未到位 -> `连通阀故障`。
- 电动通气阀未打开到位 -> `左(或右)电动通气阀故障`。
- 燃油测量系统故障/降级 -> `燃油测量系统故障`。

### 加油流程
3. 若0号油箱有可外供燃油量：
   - 打开左吊舱切断阀、0号左/右加油泵切断阀；
   - 任一阀未开到位 -> 不允许自动空中加油并告警；
   - 到位后启动0号左/右加油泵（固定翼额定档/直升机低档）；
   - 若收到低压（0号左或右）-> 不允许自动空中加油；否则提示`燃油预位`。

4. 若0号油箱无可外供燃油量：
   - 打开左吊舱切断阀、2号/3号加油泵切断阀；
   - 任一阀未开到位 -> 不允许自动空中加油并告警；
   - 到位后启动2号/3号加油泵（固定翼额定档/直升机低档）；
   - 若收到低压（2号或3号）-> 不允许自动空中加油；否则提示`燃油预位`。

5. 接收吊舱控制装置信息，进入加油过程，实时采集：
   - 燃油测量系统油量信息；
   - 机电系统加油泵低压信号。

6. 运行中切换逻辑：
   - 若收到`0号左加油泵低压`或`0号右加油泵低压`：
     - 启动2/3号路径（切断阀+加油泵）；
     - 关闭0号左/右路径（切断阀+加油泵）。

7. 油量平衡控制（2/3号泵工作时）：
   - 条件：左右机翼油量差 > 1000kg（固定翼）/500kg（直升机）；
   - 动作：关闭油量少侧加油泵切断阀；
   - 恢复条件：油量差 < 600kg（固定翼）/300kg（直升机）后重新打开。

8. 不平衡告警：
   - 左右机翼油量差 > 1200kg（固定翼）/600kg（直升机）
   - 告警：`加油燃油不平衡`。

### 退出条件
9. 模式指令为“无效” -> 退出左吊舱空中加油模式。

## 4.2.7.3 右吊舱空中加油模式（PROJECT-2007R_SDTD_0024）

> 与4.2.7.2基本镜像，左吊舱替换为右吊舱。

### 进入条件
1. 接收RIU“加受油模式=右吊舱”。

### 前置检查与故障处理
2. 检查项同左吊舱模式（阀位/电动通气阀/燃油测量系统）。
- 异常处理规则与超时（5s）同4.2.7.2。

### 加油流程
3~6. 与4.2.7.2一致（右吊舱路径），包含：
- 0号油箱有油/无油分支；
- 阀到位检查；
- 低压判定与禁止自动加油条件；
- 0号路径低压触发向2/3号路径切换。

### 平衡与告警
7. 油量平衡阈值：1000/500 触发，600/300恢复。
8. 不平衡告警阈值：1200/600。

### 退出条件
9. 模式指令为“无效” -> 退出右吊舱空中加油模式。

## 4.2.7.4 左右吊舱空中加油模式（PROJECT-2007R_SDTD_0025）

### 进入条件
1. 接收RIU“加受油模式=左右吊舱”。

### 前置检查
2. 与单吊舱模式相同（压力加油控制活门/三通阀/应急放油切断阀/连通阀/电动通气阀/燃油测量系统）。
- 失败处理同单吊舱：故障提示+必要时5s重试逻辑。

### 加油流程
3. 打开左右吊舱切断阀 + 2号阀 + 0号左/右阀 + 3号阀。
- 任一未到位 -> 不允许自动空中加油并告警。
- 到位后启动2号、0号左、0号右、3号加油泵（固定翼额定档/直升机低档）。
- 任一低压（2/0左/0右/3）-> 不允许自动空中加油。
- 全部无低压 -> 提示`燃油预位`。

4. 接收吊舱控制装置信息，进入加油过程，实时采集油量与低压。

5. 油量平衡控制：阈值同前（1000/500触发，600/300恢复）。

6. 不平衡告警：阈值同前（1200/600）。

### 退出条件
7. 模式指令“无效” -> 退出左右吊舱模式。

## 4.2.7.5 空中受油模式（PROJECT-2007R_SDTD_0026）

### 进入条件
1. 接收RIU“模式=空中受油”。

### 前置准备与检查
2. 预置动作：
- 压力加油控制活门上电；
- 三通阀置空中受油位/关闭位。

2.1 检查项：
- 压力加油控制活门上电状态；
- 三通阀到位；
- 加油泵切断阀关位；
- 吊舱切断阀关位；
- 应急放油切断阀关位；
- 连通阀关位；
- 电动通气阀开位；
- 燃油测量系统正常。

2.a~2.h 失败处理（含5s重试项）：
- 活门未上电 -> `XX油箱压力加油控制活门故障`；
- 三通阀异常 -> `三通阀故障`；
- 加油泵切断阀/吊舱切断阀/应急放油切断阀/连通阀未到位：发控制指令，5s超时告警；
- 电动通气阀异常 -> `电动通气阀故障`；
- 燃油测量系统组合故障条件触发 -> `燃油测量系统故障`。

### 预设受油量与分配
3. 前置检查正常后，向RIU发送`预设受油量`请求。

4. 接收预设受油量并分配：
- 预设受油量需满足：`> 当前总油量` 且 `<= 33000kg`，否则`预设受油失败`。
- 分配分支：
  - `预设 <= 16400kg`：
    - 0号油箱保留现有油量；
    - 1/4号油箱油量值 = `(预设总油量 / 2 - 0号油箱现有油量值) / 2`；
    - 2/3号油箱油量值 = `预设总油量 / 4`；
    - 若仅一个油箱现有油量大于初始分配值，则按下述规则重分配：
      - `a)` 若1号油箱现有油量大于分配油量，则1号油箱保留现有油量，4号油箱预设油量等于1号油箱现有油量，2号和3号油箱油量 = `(预设总油量 - 0号油箱现有油量值 - 1号油箱现有油量 * 2) / 2`；
      - `b)` 若2号油箱现有油量大于分配油量，则2号油箱保留现有油量，3号油箱预设油量等于2号油箱现有油量，1号和4号油箱油量 = `(预设总油量 - 0号油箱现有油量值 - 2号油箱现有油量 * 2) / 2`；
      - `c)` 若3号油箱现有油量大于分配油量，则3号油箱保留现有油量，2号油箱预设油量等于3号油箱现有油量，1号和4号油箱油量 = `(预设总油量 - 0号油箱现有油量值 - 3号油箱现有油量 * 2) / 2`；
      - `d)` 若4号油箱现有油量大于分配油量，则4号油箱保留现有油量，1号油箱预设油量等于4号油箱现有油量，2号和3号油箱油量 = `(预设总油量 - 0号油箱现有油量值 - 4号油箱现有油量 * 2) / 2`；
    - 若出现两个油箱或两个以上油箱现有油量大于初始分配值 -> `预设受油失败`。
  - `预设 > 16400kg`：
    - 1/4号油箱油量值 = `4100kg`；
    - 2/3号油箱油量值 = `预设油量 / 4`；
    - 0号油箱油量值 = `预设油量 / 2 - 8200kg`；
    - 若仅一个油箱现有油量大于初始分配值，则按下述规则重分配：
      - `a)` 若2号油箱现有油量大于分配油量，则2号油箱保留现有油量，3号油箱预设油量等于2号油箱现有油量，1号和4号油箱油量 = `4100kg`，0号油箱油量 = `预设总油量 - 8200kg - 2号油箱现有油量 * 2`；
      - `b)` 若3号油箱现有油量大于分配油量，则3号油箱保留现有油量，2号油箱预设油量等于3号油箱现有油量，1号和4号油箱油量 = `4100kg`，0号油箱油量 = `预设总油量 - 8200kg - 3号油箱现有油量 * 2`；
      - `c)` 若0号油箱现有油量大于分配油量，则0号油箱保留现有油量，1号和4号油箱油量 = `4100kg`，2号和3号油箱油量 = `(预设总油量 - 0号油箱现有油量值 - 8200kg) / 2`；
    - 若出现两个油箱或两个以上油箱现有油量大于初始分配值 -> `预设受油失败`。

5. 若某油箱现有油量 > 分配值：关闭该油箱压力加油控制活门。

6. 收到RPDU“需关闭活门下电”信号：发送`受油正常`。

### 受油过程闭环
7. 实时比较油量：
- 某油箱达到预设或满油量 -> 控制RIU关闭该油箱活门；
- 10s内收到关闭到位 -> 该油箱停止受油；全部到位后提示`受油完成`；
- 10s未到位 -> 判定活门关闭故障，断电关闭全部活门，并发送`受油关闭故障`告警。

8. 若全机总油量达到预设受油量：
- 断电关闭全部压力加油控制活门；
- 发送`受油完成`至RIU；
- 10s后控制三通阀关闭，终止受油。

### 故障处置
9. 空中受油期间周期巡检：
- 燃油测量系统故障；
- 机电系统通信故障；
- 控制器故障；
触发`受油故障`告警。

9.2 单油箱高油面信号触发：
- 关闭三通阀；
- 发`受油关闭故障`。

9.3 (1+2)与(3+4)油箱总量差达到1200kg：
- 关闭三通阀；
- 发`燃油不平衡`告警。

### 退出条件
10. 模式指令“无效” -> 退出空中受油工作模式。

## 开发落地建议（用于代码实现）

### A. 状态机主状态
- `STANDBY`
- `REFUEL_LEFT_POD`
- `REFUEL_RIGHT_POD`
- `REFUEL_BOTH_PODS`
- `AIR_RECEIVE`
- `FAULT_LOCKED`（不可恢复故障可转入）

### B. 每个模式统一子状态
- `ENTRY_CHECK`
- `PREPARE_PATH`
- `PREPOSITION_OK`（燃油预位）
- `RUNNING`
- `BALANCE_CONTROL`
- `EXITING`
- `ABORTED`

### C. 通用定时器
- 阀位到位超时：`5s`
- 活门关闭确认超时：`10s`
- 周期巡检周期：按BIT周期配置

### D. 通用事件接口（建议）
- RIU输入：模式、阀位到位、低压、通信状态
- RPDU输入：活门上电/下电请求
- 油量系统输入：各油箱油量、高油面、传感器/转换盒故障
- 输出命令：阀开/关、泵启/停、告警上报、提示信息

### E. 阈值参数化（建议配置化）
- 油量差控制触发：固定翼1000kg / 直升机500kg
- 油量差控制恢复：固定翼600kg / 直升机300kg
- 不平衡告警：固定翼1200kg / 直升机600kg
- 受油总量上限：33000kg
